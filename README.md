# 🎮 Virtual Gaming Platform (VGP)

A modern virtual gaming and rewards platform built with vanilla JavaScript, HTML, and CSS.

## 📋 Features

- 🎮 **Multiple Games**: Dice, Coin Flip, Spin Wheel
- 💰 **Wallet System**: Virtual currency management
- ⭐ **XP & VIP System**: Level up and unlock rewards
- 👥 **Referral Program**: Earn through referrals
- 📊 **Leaderboards**: Compete with other players
- 🔐 **User Authentication**: Secure login/register

## 🚀 Quick Start

1. Clone the repository
```bash
git clone https://github.com/azibonbrhexa/Happy-Earn.git
cd Happy-Earn
```

2. Open in browser
```bash
Open index.html in your web browser
```

3. Create an account
```bash
Click "Register" and create your account
```

## 🎮 Games

### Dice
Predict the number that will be rolled.

### Coin Flip
Guess heads or tails.

### Spin Wheel
Spin the wheel and win big multipliers.

## 💳 Wallet System

- Starting balance: 0 Virtual Currency
- Minimum bet: 10 VC
- Maximum bet: 100,000 VC
- Win multipliers vary by game

## ⭐ VIP Levels

1. **Bronze** (0 XP) - Base level
2. **Silver** (1,000 XP) - 10% bonus
3. **Gold** (5,000 XP) - 20% bonus
4. **Platinum** (10,000 XP) - 30% bonus

## 🔐 Security

- Client-side authentication using localStorage
- Session-based user management
- Password hashing (basic implementation)
- CORS-friendly design

⚠️ **Note**: This is a demo app. For production, use:
- Server-side authentication
- bcrypt.js for password hashing
- Database for persistent storage
- HTTPS for secure communication
